AI-Based Hiring Prediction System

An end-to-end Machine Learning project that predicts whether a candidate will be hired or rejected based on resume data.

🔹 Project Overview

This project simulates an AI-powered resume screening system used in HR analytics. It processes structured and unstructured resume data to predict hiring decisions.

🔹 Dataset

1,000+ synthetic resumes

Text Features: Skills, Certifications, Job Role

Structured Features: Experience, Salary Expectation, Projects, Education

Target: Recruiter Decision (Hire = 1, Reject = 0)

🔹 Techniques Used

Text Preprocessing

TF-IDF Vectorization

Feature Engineering

Logistic Regression

Random Forest

SVM

KNN

Model Evaluation (Accuracy, Precision, Recall, F1-Score)

🔹 Best Model

Logistic Regression achieved 97.5% accuracy.

🔹 Tech Stack

Python, Scikit-learn, Pandas, NumPy, TF-IDF, Google Colab
